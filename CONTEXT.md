# PixelLab Forge

PixelLab MCP 위에 얹는 재사용 캐시 + 비용 절감 워크플로 레이어. 게임 픽셀아트 에셋을 "새로 만들기 전에 이미 만든 걸 재사용"하게 만든다.

## Language

**스타일 앵커 (Style Anchor)**:
한 게임의 모든 생성물이 `style_images` 로 공유하는 대표 참조 이미지 묶음 (각 256px 이하).
_Avoid_: 레퍼런스 이미지(출처 원본을 가리킬 때만), 스타일 가이드(텍스트 지침을 가리킬 때만)

**아키타입 (Archetype)**:
같은 개념·같은 스타일을 공유하는 자산들을 하나로 묶은 단일 생성 슬롯.
_Avoid_: 배리에이션 그룹, 계열

**재사용 임계값 (REUSE_THRESHOLD)**:
`find` 유사도 score 가 이 값(0.6) 이상이면 생성 대신 재사용을 권장하는 경계값.
_Avoid_: 매칭 점수, 컷오프

**전역 라이브러리 (Global Library)**:
플러그인 리포 `library/` 에 git 으로 커밋·공유되는 공용 재사용 캐시 계층.
_Avoid_: 글로벌 캐시(플러그인 설치 캐시 경로와 혼동 주의)

**프로젝트 캐시 (Project Cache)**:
소비 게임 프로젝트의 `.pixellab-cache/` — 전역 라이브러리를 오버라이드하는 로컬 계층.

**자체 생성 (Self-generation)**:
Claude Code 가 PixelLab 을 거치지 않고 SVG·코드로 이미지를 직접 그리는 행위.

**프로브 배치 (Probe Batch)**:
요구를 말로 좁히기 어려울 때 방향이 다른 후보들을 최소 size 로 싸게 한 팩 뽑아 보는 탐색용 생성.
_Avoid_: 시안, 드래프트

**히어로 에셋 (Hero Asset)**:
반복 노출되거나 게임의 정체성을 정의해서 채택 전 사용자 검수가 필수인 에셋 (캐릭터·보스·맵 배경·키비주얼·UI 기준 패널).
_Avoid_: 중요 에셋, 메인 에셋

**벌크 에셋 (Bulk Asset)**:
히어로 에셋이 아닌 대량 생산형 에셋 (아이콘·소품 등) — Claude 가 자동 선별하고 사후 보고한다.

## Relationships

- 하나의 게임은 하나의 **스타일 앵커**를 가진다 (앵커 = `refs/<앵커이름>/`, gitignore — 상용 게임 원본이므로 커밋·재배포 금지)
- 하나의 **아키타입**은 한 번의 생성 호출로 만들어져 여러 sprite 파일로 매핑된다
- `find` 는 **프로젝트 캐시** → **전역 라이브러리** 순으로 조회하며, id 충돌 시 프로젝트 캐시가 이긴다
- 생성된 아키타입은 **전역 라이브러리**(기본) 또는 **프로젝트 캐시**(`--scope project`)에 등록된다
- 모든 에셋은 **히어로 에셋** 또는 **벌크 에셋** 중 하나다 — 애매하면 사용자에게 질문한다
- **프로브 배치**는 본 생성 전에만 쓰이며, 채택된 방향만 본 생성으로 이어진다

## Example dialogue

> **Dev:** "이 게임 아이콘 30개 뽑을 건데, 프롬프트마다 스타일 문구를 다시 써야 해?"
> **Domain expert:** "아니 — 게임의 **스타일 앵커**를 매 호출 `style_images` 로 넣으면 문구는 내용 설명만 하면 돼. 그리고 30개를 개별로 뽑지 말고 **아키타입**으로 묶어서 한 배치로."
> **Dev:** "비슷한 게 캐시에 있다는데 그냥 써도 돼?"
> **Domain expert:** "score 가 **재사용 임계값** 이상이면 후보일 뿐이야 — 같은 앵커로 만든 건지 확인하고 채택해."

## Flagged ambiguities

- "레퍼런스"가 ① 상용 게임 원본 이미지(출처)와 ② 생성 호출에 넣는 참조 묶음을 둘 다 가리키고 있었음 — 해소: ②는 **스타일 앵커**로 부른다. ①은 앵커의 재료일 뿐이며 어떤 리포에도 커밋하지 않는다.
- "자체 생성"의 허용 범위 — 해소(2026-07-17): 게임 화면에 들어가는 픽셀아트류는 PixelLab 경유가 기본, 코드로 그릴지 애매한 경계 케이스는 조용히 자체 생성하지 말고 사용자에게 질문한다.
