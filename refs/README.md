# refs/ — 스타일 앵커 보관소 (gitignore — 이 README 만 커밋됨)

한 게임의 모든 생성물이 `style_images` 로 공유하는 **스타일 앵커** 이미지들을 여기에 둔다.

```
refs/
├── README.md            ← 이 파일만 커밋됨
└── <앵커이름>/           ← 게임(프로젝트)당 하나, kebab-case
    ├── ANCHOR.md        ← 출처·크롭 메모(선택)
    └── *.png            ← 참조 이미지 (각 256px 이하)
```

## 규칙 (ADR-0001)

- **커밋 금지.** 앵커 재료가 상용 게임 스크린샷/스프라이트일 수 있으므로 `refs/` 는 gitignore 된다. 여기 있는 이미지를 `library/` 나 다른 리포로 복사·커밋하지 않는다 — 그건 재배포다.
- **256px 이하.** PixelLab `style_images` 는 장당 최대 256×256px. 스크린샷은 대표 부분을 크롭/축소해서 저장한다.
- **출력 size 주의.** `create_1_direction_object` 는 `style_images` 를 주면 `size` 파라미터를 무시하고 **가장 큰 참조 이미지의 크기가 출력 크기**가 된다. 목표 출력 크기에 맞춘 사본을 만들어 투입한다.
- **장수 제한.** 출력 size ≤85px → 최대 8장, ≤170px → 4장, 그 외 → 1장.
- **캐시 메타에는 앵커 이름만.** `add --anchor <앵커이름>` 으로 기록하고, 원본 이미지는 캐시에 넣지 않는다.
- **복제 금지.** 산출물이 특정 상용 게임의 알아볼 수 있는 캐릭터/에셋 근사 복제면 채택하지 않는다.
